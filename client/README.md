# Cursed Mage Website

## Local Development
 
#### Install yarn via npm
```bash
npm install -g yarn
```

#### Install dependencies
```bash
yarn
```

#### Run the development server
```bash
yarn dev
```

#### Preview the website on local network
```bash
yarn preview
```

## Deployment

#### Build the website
```bash
yarn build
```

#### Deploy the website
```bash
yarn preview
```
